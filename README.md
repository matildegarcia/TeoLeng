# Regex Lab 

Trabajo práctico enfocado en el uso de expresiones regulares (regex) para procesar texto en Python.

Se trabaja con:

* Facturas en PDF
* Un estado bancario en XML

Objetivo:
Extraer información, validar coincidencias (fecha e importe) y modificar el XML según corresponda, utilizando únicamente regex (sin parsers de XML ni lógica tradicional).

Incluye 6 programas:

1. Lectura de PDF
2. Extracción de fecha y monto
3. Extracción de ítems
4. Lectura de XML
5. Validación de coincidencias
6. Eliminación de movimientos en el XML

Enfocado en procesamiento de texto y manejo de patrones.
